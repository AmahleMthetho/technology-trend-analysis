#  📊 Technology Trends Analysis
## 📖 Overview

This project explores global technology trends using survey data collected from software developers worldwide. The objective was to transform raw survey data into meaningful business insights through data cleaning, visualization, and interactive dashboard development.

The analysis identifies current technology adoption, future technology preferences, salary trends, job demand, and demographic insights to support strategic decision-making within the technology industry.

## 🎯 Business Problem

Technology is constantly evolving, making it difficult for organisations and professionals to understand which programming languages, databases, cloud platforms, and frameworks are gaining or losing popularity.

This project analyses survey data to identify industry trends and provide data-driven recommendations for businesses, recruiters, and technology professionals.

## 🔄 Methodology

The project followed an end-to-end data analysis process:

Imported survey datasets
Cleaned and transformed the data using Power Query
Standardised data fields
Performed exploratory data analysis
Built interactive dashboards in IBM Cognos Analytics
Identified trends and generated business insights

##  💡  Key Findings
Programming Languages
JavaScript and SQL remain among the most widely used technologies.
Go demonstrates strong future growth despite lower current adoption.

Databases
PostgreSQL remains the leading database.
Redis shows significant future growth potential.

Cloud Platforms
Amazon Web Services (AWS) continues to dominate cloud adoption.

Demographics
Developers aged 25–34 represent the largest respondent group.
Bachelor's degree holders form the largest educational segment.

## 🚀 Business Impact

The findings can help organisations:

Identify emerging technology trends
Support recruitment strategies
Guide workforce planning
Prioritise employee upskilling
Make informed technology investment decisions
